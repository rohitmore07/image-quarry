## Project Title - Image Quarry
ImageQuarry react application allows users to search for images using the Unsplash API. When a user enters a search term in the search bar, the application sends a request to the Unsplash API and displays the matching photos in a four-column grid layout.

## Unsplash API Integration
This application integrates with the Unsplash API to fetch relevant images based on the user's search query. The Unsplash API provides access to a vast collection of high-quality images, allowing users to discover and display visually appealing photos.
